# My Dotfiles

## How to apply

Use `stow -v .` to symlink these dotfiles to your home directory.

## Application List

| Type | Choice | Package(s) |
|-|-|-|
| Window Manager | Hyprland | `hyprland` |
| Greeter | SDDM | `sddm` |
| AUR Helper | Yet Another Yoghurt (yay) | `yay` |
| File Manager | Nautilus | `nautilus` |
| Browser | Firefox | `firefox` |
| Simple Text Editor | Sublime Text | `sublime-text` |
| Code Editor / IDE | Visual Studio Code | `visual-studio-code` |
| Terminal Emulator | Kitty | `kitty` |
| Video Player | MPV / VLC | `mpv`, `vlc` |
| Status Bar | Waybar | `waybar` |
| App Launcher | Rofi | `rofi` |
| Clipboard | wl-clipboard + cliphist |  `cliphist`, `wl-clipboard` |
| Screen Locker | Hyprlock | `hyprlock` |
| System Monitor | Btop / Mission Center | `btop`, `mission-center` |
| Color Palette | Matugen | `matugen` |
| Wallpaper picker | waypaper | `waypaper` |
| Notification Daemon | swaync | `swaync` |
